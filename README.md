# Sistema de Gestión de E-commerce

## Descripción

Este proyecto consiste en una API REST desarrollada en Go para la administración de productos de una tienda virtual. Permite realizar operaciones de consulta, creación, actualización y eliminación de productos utilizando programación orientada a objetos.

## Tecnologías utilizadas

- Go
- Visual Studio Code
- GitHub
- HTTP
- JSON

## Funcionalidades

- Listar todos los productos.
- Obtener un producto por ID.
- Agregar un producto.
- Actualizar un producto.
- Eliminar un producto.
- Buscar productos por nombre.
- Mostrar productos con stock disponible.
- Mostrar productos con stock bajo.

## Estructura del proyecto

- controllers
- routes
- services
- models
- database
